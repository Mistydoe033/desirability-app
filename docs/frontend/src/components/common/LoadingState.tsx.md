**What It Does**
- Reusable animated loading component with accessible status text.
- Uses MUI and keyframes for consistent styling.

**Why This Structure**
- Avoids inconsistent loading UI across pages.
- Encapsulates animation details in one component.

**Principles**
- Reusable UI primitives.
- Accessibility-aware feedback.


**Principles Explained**
- `Reusable UI primitives`: Create common building blocks for consistent UI behavior.
- `Accessibility-aware feedback`: Make loading/errors perceivable for assistive technologies.
