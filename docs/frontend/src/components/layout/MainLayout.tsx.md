**What It Does**
- App layout with navigation, responsive drawer, and content container.
- Provides shared app chrome for all routes.

**Why This Structure**
- Keeps layout concerns separate from page content.
- Improves UX consistency across routes.

**Principles**
- Layout composition.
- Responsive design.


**Principles Explained**
- `Layout composition`: Wrap pages in a shared layout to keep navigation and structure consistent.
- `Responsive design`: Ensure layouts adapt to different screen sizes and orientations.
