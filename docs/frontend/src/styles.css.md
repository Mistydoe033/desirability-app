File: [`frontend/src/styles.css`](../../../frontend/src/styles.css)

**What It Does**
- Global styles, font imports, and CSS variables.
- Defines background gradients and Leaflet container styling.

**Why This Structure**
- Keeps global styles separate from component styles.
- Ensures consistent base look and map rendering.

**Principles**
- Global theming.
- Accessibility-friendly focus styling.


**Principles Explained**
- `Global theming`: Centralize colors/typography so the UI stays consistent.
- `Accessibility-friendly focus styling`: Provide visible focus indicators for keyboard users.
