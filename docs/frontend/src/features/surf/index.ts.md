**What It Does**
- Barrel export for surf feature components.
- Makes surf tooling easy to import into Details page.

**Why This Structure**
- Keeps feature boundaries explicit.
- Simplifies imports.

**Principles**
- Encapsulation.
- Clear feature API.


**Principles Explained**
- `Encapsulation`: Expose a small public API while hiding implementation details to reduce coupling.
- `Clear feature API`: Expose a small, well-defined interface for the feature.
