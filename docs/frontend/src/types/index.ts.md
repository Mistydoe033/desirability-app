**What It Does**
- Barrel export for UI and API types.
- Simplifies imports from `types/`.

**Why This Structure**
- Provides a single entrypoint for type definitions.
- Keeps code using consistent import paths.

**Principles**
- Encapsulation.
- Consistency.


**Principles Explained**
- `Encapsulation`: Expose a small public API while hiding implementation details to reduce coupling.
- `Consistency`: Use consistent conventions to reduce cognitive load and avoid surprises.
