File: [`frontend/src/pages/Home.tsx`](../../../../frontend/src/pages/Home.tsx)

**What It Does**
- Landing page with product positioning and quick actions.
- Navigates to Search or Browse flows.

**Why This Structure**
- Gives users a clear starting point and story.
- Keeps page-level content isolated from reusable components.

**Principles**
- Clear user journey.
- Separation of page-level concerns.


**Principles Explained**
- `Clear user journey`: User flow is obvious from entry point to key actions.
- `Separation of page-level concerns`: Keep page-level concerns in separate modules so each part stays focused and changes don’t ripple.
